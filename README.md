# Hiçkorkmaz Garaj V7

GitHub Pages'e yükleme:

1. GitHub reposunu aç.
2. Bu paketin içindeki dosyaları ana klasöre yükle:
   - index.html
   - style.css
   - app.js
   - manifest.json
3. Commit changes de.
4. Settings > Pages bölümünde Branch: main / root seçili olsun.
5. Siteyi yenile.

V7 özellikleri:
- Dashboard
- Müşteri/Firma bazlı toplam borç
- Plaka bazlı borç
- İsim/firma/plaka arama
- İsim/firma aranınca kayıtlı tüm plakaları listeleme
- Plaka aranınca direkt araç geçmişi
- Servis geçmişi
- Tahsilat geçmişi
- Borç takibi
- Mobil uyumlu koyu tema

Not:
Veriler localStorage içinde saklanır. Aynı cihaz/tarayıcıda kalır.
